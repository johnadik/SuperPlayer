basic usage

Using SuperPlayer is very easy. To use SuperPlayer, you need to include jQuery, superplayer.js/superplayer.min.js, and superplayer.css/superplayer.min.css

Or

Do not use both superplayer.js and superplayer.min.js - Only use one.

Do not use both superplayer.css and superplayer.min.css - Only use one.
